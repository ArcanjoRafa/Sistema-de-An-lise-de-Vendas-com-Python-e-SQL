Este projeto consiste em um sistema de análise de vendas, desenvolvido em Python, utilizando SQLite como banco de dados e Pandas + Matplotlib para análise e visualização dos dados.

O sistema permite:
  Armazenar dados de vendas em banco relacional
  Realizar consultas SQL com agregações
  Analisar dados utilizando Pandas
  Gerar gráficos para apoio à tomada de decisão
  Projeto desenvolvido com foco em portfólio para análise de dados.

Tecnologias Utilizadas:
  Python
  SQLite
  Pandas
  Matplotlib
  SQL
  Git

OBS: Este projeto faz parte do meu processo de aprendizado em Análise de Dados e está em constante evolução.
